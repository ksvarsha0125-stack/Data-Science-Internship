#  Iris Flower Classification

## Internship

##  Problem Statement
The objective of this project is to classify Iris flowers into
Setosa, Versicolor, and Virginica using machine learning algorithms
based on their physical features.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

##  Methodology
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training using KNN
6. Prediction and Evaluation

##  Model Used
- K-Nearest Neighbors (KNN)

## Evaluation Metric
- Accuracy Score

##  Conclusion
The model successfully classifies Iris flower species with high accuracy.
