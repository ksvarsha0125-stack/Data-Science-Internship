## Titanic Survival Prediction

##  Internship
CodSoft Data Science Internship

## Problem Statement
The goal of this project is to predict whether a passenger survived the
Titanic disaster based on passenger features using machine learning
techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

##  Methodology
1. Data Collection
2. Data Cleaning
3. Encoding Categorical Variables
4. Train-Test Split
5. Model Training using Logistic Regression
6. Prediction and Evaluation

##  Model Used
- Logistic Regression

## Evaluation Metric
- Accuracy Score

##  Conclusion
The Logistic Regression model predicts passenger survival with good
accuracy based on selected features.
